# TDSEWorkshop

A lightweight Java project providing mathematical utilities and user management services.

## Description

This project implements basic **mathematical calculations** and **user management** services, including operations such as square calculations and handling user information.

## Prerequisites

- **Java 21** or higher
- **Maven 3.6+**

## Installation

1. Clone or download the project
2. Navigate to the project folder
3. Install dependencies:

```bash
mvn clean install
```

## Project Structure

```
src/
├── main/java/co/edu/escuelaing/services/
│   ├── MathServices.java    # Mathematical services
│   └── User.java             # User model
└── resources/
    └── application.properties
```

## Features

- **getSquare()**: Calculates the square of an integer
- **getUser()**: Returns formatted user information (name and email)
- **postUser()**: Returns the user information sent
